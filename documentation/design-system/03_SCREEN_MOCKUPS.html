    States: - Loading: Skeleton cards (2 per column) with shimmer animation - Empty column: Dashed border zone with "Drop deals here" - Empty pipeline: "Your pipeline is empty. [Import from CSV] or [Add your first deal]" - Drag-over: Column border highlights primary-400, insertion line shows between cards - Multi-select: Cmd+click cards, blue selection border, drag batch

    Responsive: - 768-1023px: 2-3 columns visible, horizontal scroll for rest - <768px: Single column with swipe to switch stages, "+" FAB to add deal - Card adapts: Shows owner avatar + deal name + value only on mobile, expandable
