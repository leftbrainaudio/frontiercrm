    FrontierCRM — Salesforce Sales Cloud Competitive Research Report

    Sources Used

    1. Wikipedia (Salesforce page - Services, Platform, AI sections)
    2. Salesforce product pages - salesforce.com/sales/cloud/ and salesforce.com/sales/ (structured data and FAQ content)
    3. r/salesforce community posts (Summer 26 / Winter 26 Release Notes abridged, consultant posts, user complaints)

    1. KEY FEATURES BY CATEGORY

    Contact Management

    • Centralized contact/account/lead database with 360-degree view
    • Activity history tracking (calls, emails, meetings)
    • Einstein Activity Capture for email-to-contact linking
    • Duplicate management and fuzzy matching
    • Free CRM capped at 2 users; Starter Suite at $25/user/mo
    • Contact enrichment requires Data Cloud credits (metered consumption)

    Deal Pipeline

    • Visual sales pipeline - stage-based opportunity tracking
    • Kanban board view and list view
    • Opportunity splits, multi-currency, discounting, quoting
    • Einstein forecasting (predictive AI on win probability)
    • CPQ module and Revenue Lifecycle Management
    • Forecasting is complex to configure; basic pipeline needs Enterprise ($175/user/mo)

    Email Integration

    • Outlook/Gmail integration via Einstein Activity Capture (EAC)
    • Lightning Sync being retired (2027); EAC is the replacement
    • Microsoft retiring EWS forces Graph API migration
    • Community quote: "The year is 2027 and we are still figuring out email sync"
    • DKIM/SPF enforcement for unverified domains

    Reporting and Analytics

    • Reports with drag-and-drop builder
    • Dashboards with up to 20 components
    • Tableau (acquired $15.7B) - embedded analytics; Tableau Next still "work in progress"
    • Einstein Analytics / CRM Analytics
    • Excel/CSV export with formula sanitization
    • Advanced analytics requires Tableau license or Data Cloud consumption

    Automation (Sales Force Automation / SFA)

    • Flows (primary automation tool - screen, auto-launched, scheduled, record-triggered)
    • Flow Orchestration (now free - was paid)
    • Flow version comparison tool
    • Approval processes (Flow-based)
    • Agentforce: Autonomous AI agents across deal cycle
    • Einstein GPT for automated content generation
    • Powerful but steep learning curve - 6-12 months to proficiency

    Lead Scoring

    • Einstein Lead Scoring (AI-predicted conversion probability)
    • Foundations Lead Scoring (new Summer 26) - rules + models via MCNext
    • Rules-based lead assignment + round-robin
    • AI scoring is an upsell at higher tiers

    Mobile

    • Salesforce Mobile App (iOS/Android)
    • Tableau Next Dashboards in mobile
    • React Apps with Agent SDKs
    • Actionable Push Notifications (GA)
    • Admin-customized home page
    • Mobile app is functional but often slower than native CRM apps

    Collaboration

    • Slack integration - Salesforce channels auto-created in Enterprise/Unlimited
    • Salesblazer Community (Slack-based)
    • ChatGPT integration in Slack
    • Quip for document collaboration
    • Chatter being deprecated
    • Slack integration is Enterprise-tier only

    Customization

    • Force.com PaaS - custom objects, fields, record types, layouts
    • Apex (Java-like proprietary language)
    • Lightning Web Components (LWC) - modern UI framework
    • Flows for declarative automation
    • AppExchange - 1500+ third-party apps
    • Multi-Framework - now supports React apps
    • MuleSoft for integration
    • Proprietary tech stack creates vendor lock-in

    Integrations

    • MuleSoft Anypoint Platform
    • Informatica (acquired ~$8B) - data management
    • Tableau ($15.7B) - visual analytics
    • Slack ($27.7B) - team collaboration
    • Contentful (acquired June 2026, $1-1.5B) - content management
    • Data Cloud - real-time data unification (metered consumption)
    • REST/SOAP APIs, Bulk API, Streaming API, GraphQL
    • 4000+ integrations on AppExchange
    • MCP compatibility for AI tools
    • Most powerful integrations cost extra at higher tiers

    2. TOP 7 USER COMPLAINTS / PAIN POINTS

    1. Pricing Complexity and Hidden Costs

    • Base price ($25-$550/user/mo) is just the entry point
    • Data Cloud consumption credits are "impossible to forecast and impossible to cap"
    • Marketing Cloud Next is "metered against Data Cloud credits no one can forecast"
    • Agentforce billing per conversation/token is opaque
    • Feature retirements force expensive migrations
    • Community: "The rebrand is cosmetic, but it quietly walks you onto consumption billing"

    2. Complexity and Steep Learning Curve

    • Real proficiency takes 6-12 months
    • "Every client has experienced a disaster implementation" - consultant
    • Requires dedicated administrators or expensive consultants
    • Apex proprietary language creates vendor lock-in
    • Release cycles (3x/year + monthly security patches) force constant changes
    • SMBs cannot afford a full-time admin

    3. Feature Retirement and Forced Migration Fatigue

    • Process Builder - to be deprecated, rebuild in Flows
    • Chatter - being deprecated
    • SFO / Lightning Sync - retiring 2027
    • Desk.com - consolidated into Service Cloud
    • Visualforce - being replaced by LWC
    • Developer Console - being replaced by Web Console
    • Custom buttons/links - being replaced by LWC actions
    • Frequent forced migration creates massive overhead

    4. Email Integration Is Perpetually Broken

    • "The year is 2027 and we are still figuring out email sync" - community quote
    • Microsoft retiring EWS forces Graph API migration
    • Lightning Sync and SFO both being retired
    • EAC threaded view "not available (Soon trademark)"
    • Email is the most fundamental CRM function and it remains unreliable

    5. Breaking Changes with Short Migration Windows

    • "Monthly Releases was just an excuse for Massive Shitshow" (Summer 26 notes)
    • One-month enforcement timelines for breaking changes
    • Non-enterprise VPNs blocked with short notice
    • Release updates break old packages and integrations
    • Certificate changes impact integrations with frequent rotations
    • SMBs without dedicated admin staff are hit hardest

    6. AI Is an Upsell, Not a Feature

    • Einstein/Agentforce requires higher-tier licenses or add-on consumption
    • "Make the platform plumbing frictionless so you wander into the Agentforce and Data Cloud turnstiles where the meter actually runs"
    • Agentforce for Flow "isn't ready for Prod but you can spend an unknown amount of tokens"
    • Non-English languages = 50% token cost increase
    • AI features exist but are priced for enterprise

    7. Security Overreach with Blunt Instruments

    • Non-enterprise VPNs blocked (NordVPN etc.)
    • "Improving security so tight that guidance was you will need per-device hardware or biometric authentication"
    • Email-only logins as default
    • Certificate validation with frequent rotations breaks integrations
    • Community: "slightly hindering attackers is worth alienating most your client base"

    3. GAPS / INNOVATIONS NO CRM DOES WELL

    Industry Gaps (Opportunities for FrontierCRM)

    1. Pricing Transparency and Predictability - Every major CRM has opaque pricing, hidden add-ons, and consumption-billing surprises - Opportunity: Flat, predictable per-user pricing with ALL features included. No credits, no consumption bills.

    2. True Email Integration (Not Perpetual Beta) - Salesforce has been "figuring out email sync" for 20+ years - HubSpot has decent sync but limited depth - Opportunity: Bulletproof, native two-way email sync. Logging, templates, sequences, analytics - included.

    3. SMB-First Simplicity - Major CRMs are either too simple (Pipedriver) or too complex (Salesforce) - Opportunity: The Goldilocks CRM - powerful enough to grow into, simple enough to set up in an afternoon. No proprietary language.

    4. Predictable, Non-Breaking Releases - Salesforce pushes breaking changes 3x/year with month-long enforcement windows - Opportunity: Backward-compatible, opt-in upgrade model. Let users choose when to adopt new features.

    5. Built-in Lead Scoring Without AI Upsells - Every CRM gates lead scoring behind AI add-ons or enterprise tiers - Opportunity: Simple, transparent lead scoring based on activity, engagement, and fit - included at every tier.

    6. Real Mobile-First Experience - Salesforce mobile is a web wrapper - Opportunity: Native mobile-first architecture. Full-featured mobile app that doesn't feel like a downgrade.

    7. Native Customer Portal / Self-Service - Salesforce requires Experience Cloud (separate license) for portals - Opportunity: Native customer portal included at every tier.

    Pro Suite $100 Growing SMBs
    Enterprise $175 Mid-market
    Unlimited $350 Large enterprises
    Agentforce 1 Sales $550 Full AI-powered enterprise

    Note: List prices before discount. Enterprise deals are 20-40% off through negotiation. SMBs pay list price.

    Strategic Summary for FrontierCRM

    Salesforce's key vulnerability for SMBs: 1. Complexity requiring consultants or dedicated admins 2. Pricing opacity - base price is 30% of actual total cost 3. Breaking changes with short migration windows 4. Email sync that has never fully worked 5. AI features dangled but priced for enterprise 6. Consumption billing that cannot be forecast 7. Forced migrations when features are retired

    FrontierCRM should position against these specific pain points: - Simple onboarding (productive in 15 minutes, no consultant needed) - All-inclusive pricing (no consumption credits, no hidden tiers) - Reliable email sync (works out of the box, always) - Backward-compatible upgrades (opt-in, no breaking changes) - AI included at every tier (not upsold) - Mobile-first architecture (full parity with desktop) - Native customer portal (no extra license required)
