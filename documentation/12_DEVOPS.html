    FrontierCRM — DevOps & Agile Workflow

    How we build, review, and ship. Managed via Hermes kanban board with ALLSTARS profiles.

    Sprint Cadence

    Ceremony When What
    Sprint planning Monday AM Move tasks from backlog → sprint on kanban. Assign to profiles.
    Daily standup Daily check Coach checks board, unblocks blockers, reassigns stuck tasks.
    Sprint review Friday PM Demo completed work. User signs off or requests changes.
    Retro Friday EOD What worked, what didn't, one change for next sprint.

    Cadence: 1-week sprints. Each sprint = a numbered S0, S1, S2, etc.

    Kanban Workflow

    Backlog → Triage → Ready → Running → Review → Done
    Stage What happens Owner
    Backlog Ideas, not yet scoped Anyone
    Triage Spec needs writing Atlas
    Todo Ready but waiting on dependencies Coach
    Ready Claimable by assignee profile Dispatcher
    Running Worker actively executing Assignee
    Blocked Stuck on dependency, needs input, or needs review Assignee + Coach
    Done Verified and signed off User

    CI Pipeline

    Planned (not yet implemented):

    Step Tool Runs on
    Lint ruff (Python), oxlint (JS) Pre-commit + CI
    Type check pyright (Python), tsc --noEmit (TS) CI
    Test pytest (Python), vitest (React) CI
    Build vite build CI

    Merge gating: Code review by allstars-auditor profile required before merge.

    Profiles & What They Own

    Profile Role Scope
    Atlas Architect Specs, ADRs, system design, architecture decisions
    Builder Implementer Code, diffs, builds, commands
    Prober Tester Test coverage, repro cases, QA
    Detective Debugger Root-cause analysis, bug reproduction
    Auditor Reviewer Code review, security audit, severity reports
    Creative Designer Logo, UI mockups, brand assets, visual QA
    Scout Researcher Market research, competitive analysis, tech landscape
    Scribe Writer READMEs, runbooks, docs, release notes
    Operator DevOps Deploys, infra, monitoring, rollbacks
    Coach PM/Orchestrator Board management, task routing, unblocking
