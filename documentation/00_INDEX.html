    Product Vision

    One sentence: A CRM that SMB-to-mid-market teams actually want to use — beautiful, intelligent, integrated, and transparent.

    Core tenets: 1. Beautiful UX — Modern, light, efficient design that makes daily CRM use a pleasure, not a chore 2. AI-native — Predictive lead scoring, smart scheduling, auto-sequencing, conversational analytics — included at every paid tier 3. Deep integrations — Email, calendar, Slack, accounting, e-commerce — native, not Zapier-dependent 4. No-code workflows — Drag-and-drop automation builder for non-technical teams 5. Transparent pricing — Single per-user price, all features included, no hidden SKUs 6. Fast onboarding — Import from anywhere, setup in minutes, context-sensitive help

    Target Market

    Segment Size (US) Employees Price Target
    Primary: SMB ~30,000 companies 10–200 $19–49/user/mo
    Secondary: Mid-market ~10,000 companies 200–1,000 $49–99/user/mo
    Tertiary: Enterprise ~5,000 companies 1,000+ Custom

    Key Differentiators (Validated by Research)

    1. Conversation Intelligence — Gong/Chorus-level call analysis at SMB prices. The #1 whitespace opportunity. No competitor (HubSpot, Pipedrive, Zoho) offers native CI.
    2. AI included at every tier — Not gated behind $50–100/user/mo add-ons. Lead scoring, email drafting, meeting summarization, predictive forecasting — all in the base price.
    3. Design-first UI — Most CRMs are utilitarian. FrontierCRM treats the UI as a product. WCAG 2.1 AA verified.
    4. Integrated by default — Native Gmail/Outlook sync, Slack integration, calendar scheduling, accounting sync.
    5. Transparent pricing — Public pricing, no sales calls required to see a price, no contract lock-in.
    6. Mobile-first architecture — PWA with offline mode, full feature parity. HubSpot/Pipedrive have no offline create/edit.
    7. Headless/embeddable CRM — React component library + web components to embed CRM surfaces anywhere. No competitor offers this.
